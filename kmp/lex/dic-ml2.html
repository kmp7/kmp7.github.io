This web application is a multilingual dictionary of machine learning terminology designed specifically for linguistics students studying LLMs (Large Language Models). It provides a comprehensive collection of key terms in the field of machine learning with translations and definitions in English, Russian, and German.

Key Features

  • Multilingual Interface: The entire application interface is available in three languages: English, Russian, and German.
  • Terminology Management: Each dictionary entry contains the term and its definition in all three languages.
  • Add Custom Terms: Users can add new terms through a convenient modal form.
  • Search Functionality: Quickly find terms by typing in the search box.
  • Dark/Light Mode: Toggle between dark and light themes for comfortable reading in any environment.
  • Import/Export: Save your dictionary as a JSON file or import previously saved dictionaries.
  • Persistent Storage: Your dictionary and preferences are automatically saved in the browser's local storage.
  • Responsive Design: The application works well on both desktop and mobile devices.

This application was created as an educational tool to help linguistics students better understand the terminology used in LLM research and development. It's completely client-side with no server requirements, making it easy to use in any modern web browser.

Это веб-приложение представляет собой многоязычный словарь терминологии машинного обучения, разработанный специально для студентов-лингвистов, изучающих LLM (большие языковые модели). Оно предоставляет обширную коллекцию ключевых терминов в области машинного обучения с переводами и определениями на английском, русском и немецком языках.

Ключевые возможности

  • Многоязычный интерфейс: Весь интерфейс приложения доступен на трех языках: английском, русском и немецком.
  • Управление терминологией: Каждая словарная статья содержит термин и его определение на всех трех языках.
  • Добавление пользовательских терминов: Пользователи могут добавлять новые термины через удобную модальную форму.
  • Функция поиска: Быстрый поиск терминов путем ввода в поисковое поле.
  • Темный/светлый режим: Переключение между темной и светлой темами для комфортного чтения в любой среде.
  • Импорт/экспорт: Сохранение словаря в виде JSON-файла или импорт ранее сохраненных словарей.
  • Постоянное хранение: Ваш словарь и предпочтения автоматически сохраняются в локальном хранилище браузера.
  • Адаптивный дизайн: Приложение хорошо работает как на настольных, так и на мобильных устройствах.

Diese Webanwendung ist ein mehrsprachiges Wörterbuch für Fachbegriffe des maschinellen Lernens, das speziell für Linguistikstudenten entwickelt wurde, die LLMs (Large Language Models) studieren. Es bietet eine umfassende Sammlung von Schlüsselbegriffen auf dem Gebiet des maschinellen Lernens mit Übersetzungen und Definitionen auf Englisch, Russisch und Deutsch.

Hauptfunktionen

  • Mehrsprachige Benutzeroberfläche: Die gesamte Anwendung ist in drei Sprachen verfügbar: Englisch, Russisch und Deutsch.
  • Terminologieverwaltung: Jeder Wörterbucheintrag enthält den Begriff und seine Definition in allen drei Sprachen.
  • Benutzerdefinierte Begriffe hinzufügen: Benutzer können über ein praktisches Modalformular neue Begriffe hinzufügen.
  • Suchfunktion: Schnelles Finden von Begriffen durch Eingabe in das Suchfeld.
  • Dunkel-/Hellmodus: Umschalten zwischen dunklem und hellem Design für angenehmes Lesen in jeder Umgebung.
  • Import/Export: Speichern Sie Ihr Wörterbuch als JSON-Datei oder importieren Sie zuvor gespeicherte Wörterbücher.
  • Persistente Speicherung: Ihr Wörterbuch und Ihre Einstellungen werden automatisch im lokalen Speicher des Browsers gespeichert.
  • Responsives Design: Die Anwendung funktioniert sowohl auf Desktop- als auch auf mobilen Geräten.

О приложении

Данное приложение представляет собой визуальный словарь дефиниций термина "Нейросеть", классифицированных по категориям, уровню научности и семантической близости. Приложение позволяет исследовать различные определения и их взаимосвязи с помощью интерактивных визуализаций.

Функциональность:

  • Фильтрация определений по категориям
  • Настройка уровня научности определений
  • Поиск по ключевым словам
  • Сортировка по различным критериям
  • Два режима визуализации: круговая диаграмма и сетевой граф
  • Адаптивный дизайн для различных устройств
  • Поддержка темного и светлого режимов

Использованные технологии: HTML5, CSS3, JavaScript, D3.js v7

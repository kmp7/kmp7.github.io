О приложении

Это интерактивное веб-приложение создано для визуализации и исследования концепций искусственного интеллекта как феномена и ноумена.

Функциональность:

  • Интерактивные визуализации ключевых аспектов ИИ
  • Сравнительная таблица феномена и ноумена ИИ
  • Возможность добавления пользовательских заметок
  • Переключение между светлой и темной темами
  • Адаптивный дизайн для мобильных устройств

Использованные технологии:

  • HTML5 для структуры документа
  • CSS3 для стилизации и адаптивного дизайна
  • JavaScript для интерактивности
  • D3.js для создания динамических визуализаций
  • LocalStorage для сохранения пользовательских заметок

Morse code

Межсемиотический перевод (трансмутация) — это интерпретация знаков одной знаковой системы средствами другой. Перевод текста в азбуку Морзе представляет собой преобразование знаков алфавитной (вербально-графической) системы в знаки точечно-тире (дискретно-сигнальной) системы, где каждая буква кодируется комбинацией коротких (точка) и длинных (тире) сигналов. Обратный перевод восстанавливает текст из сигнальной последовательности.

1

Кодирование

Текст → Морзе
Точка ( · ) = короткий сигнал, тире ( — ) = длинный сигнал

Анализ перевода: Текст / Азбука Морзе

Параметр Кодирование (Текст → Морзе) Декодирование (Морзе → Текст)
Исходная знаковая система Алфавитная (буквы, цифры) ↔ Дискретно-сигнальная (точки, тире)
Целевая знаковая система Дискретно-сигнальная (точки, тире) ↔ Алфавитная (буквы, цифры)
Каналы передачи Зрительный → звуковой, световой, тактильный ↔ Звуковой, световой → зрительный
Сохранение информации Полное (взаимно однозначное соответствие) = Полное (взаимно однозначное соответствие)

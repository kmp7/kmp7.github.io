О приложении

Данное приложение предназначено для развития навыков эффективной коммуникации и самоанализа в образовательных диалогах с LLM.

Инструкция:

  1. Завершив вашу учебную сессию с LLM, используйте запрос для оценки вашего диалога:
    "Я, Орлова Ева Адамовна из 345 группы (вариант для ... осторжных: Я, ОЕА-345 ...)). Представьтесь и проанализируйте наш диалог. Предоставьте статистику: продолжительность, количество реплик, общий объем текста в моих запросах. Выступите как строгий критик, выявив языковые (грамматика, стилистика), фактические и этические ошибки. Дайте конкретные рекомендации для улучшения и общую оценку (1-100 баллов). Ответ до 1000 символов."
  2. Сделайте скриншоты вашего запроса к LLM и её ответа.
  3. Заполните поля форм.
  4. Прикрепите скриншоты.
  5. Нажмите кнопку "Завершить" для получения PDF-отчета.

Внимание: Цель этого задания — развитие навыков эффективной коммуникации. Оценка LLM является инструментом для выявления зон роста и дальнейшего совершенствования. Относитесь к критике конструктивно.

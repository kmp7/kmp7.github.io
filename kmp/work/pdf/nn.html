A

P
Perspective — Предметная область
G

T
Task — Прагматическая задача

О приложении

Данное интерактивное приложение демонстрирует концепцию полидефинитивности — подхода к построению словарных статей, при котором один термин получает множество определений в зависимости от контекста использования.

Приложение позволяет исследовать, как изменение целевой аудитории, предметной области, глубины рассмотрения, прагматической задачи и временного контекста влияет на формулировку определения термина «Нейросеть».


Лингводидактический потенциал

Полидефинитивный подход и LLM в образовании

Интеграция больших языковых моделей в образовательный процесс открывает уникальные возможности для реализации полидефинитивного подхода. LLM способны генерировать определения, адаптированные под конкретного адресата, что превращает статичный словарь в динамический инструмент персонализированного обучения.


Дидактические преимущества:

  • Развитие метаязыковой рефлексии: студент осознаёт, что определение — не догма, а коммуникативный инструмент
  • Формирование дискурсивной компетенции: умение переключаться между регистрами (научный, популярный, профессиональный)
  • Освоение терминологической вариативности: понимание, как один концепт вербализуется в разных дисциплинах
  • Критическое мышление: анализ того, что включается и исключается из определения в зависимости от контекста
  • Академическая грамотность: навык формулирования определений для разных аудиторий и целей

Интеграция с LLM:

Языковые модели позволяют масштабировать полидефинитивность: вместо конечного набора заготовленных определений система может генерировать дефиниции для любой комбинации параметров. Это создаёт среду для исследовательского обучения, где студент экспериментирует с измерениями и наблюдает, как трансформируется содержание и форма определения. Такой подход воспитывает понимание языка как гибкой системы, а не фиксированного кода.


Педагогические сценарии:

  • Задание на «перевод» определения между аудиториями (адаптация научного текста для школьников)
  • Сравнительный анализ дефиниций из разных дисциплин (междисциплинарное мышление)
  • Критическая оценка машинных определений (выявление неточностей, упрощений, смещений)
  • Создание авторских дефиниций с обоснованием выбора формулировок
  • Диахронический анализ: как менялось понимание термина во времени

Использованные технологии

  • HTML5 — семантическая разметка
  • CSS3 — стилизация, Flexbox, Grid, CSS-переменные
  • JavaScript (ES6+) — интерактивность
  • LocalStorage API — сохранение данных
  • Адаптивный дизайн — медиа-запросы

Раздел 1

Лингвистические модели

1.1. Модель фразового глагола (Phrasal Verb Model)

JavaScript

Структурная модель лингвистической единицы, представляющая фразовый глагол как объект с несколькими аспектами:

JavaScript 
// Структурная модель лингвистической единицы
{
    verb: "get up",           // Форма (морфология)
    meaning: "вставать",      // Семантика (значение)
    example: "I ___ at 7",    // Синтагматика (употребление)
    hint: "утреннее действие", // Прагматика (контекст)
    difficulty: "easy"        // Дидактическая характеристика
}

Что моделируется:

  • Морфологическая структура — глагол + частица
  • Семантическое поле — перевод, значение
  • Синтаксическая дистрибуция — позиция в предложении
  • Прагматический контекст — подсказки для понимания

Раздел 4

Алгоритмические модели

4.1. Модель случайной выборки

Алгоритм
JavaScript 
// Алгоритм Фишера-Йетса
function shuffleArray(array) {
    for (let i = array.length - 1; i > 0; i--) {
        const j = Math.floor(
            Math.random() * (i + 1)
        );
        [array[i], array[j]] = [array[j], array[i]];
    }
    return array;
}

Математическая модель: Каждая перестановка равновероятна (P = 1/n!)

4.2. Модель балансировки сложности

Алгоритм
JavaScript 
// Распределение: 30% easy, 40% medium, 30% hard
const easyCount = Math.floor(total * 0.3);
const mediumCount = Math.floor(total * 0.4);
const hardCount = total - easyCount - mediumCount;

Обеспечивает сбалансированный игровой опыт с нарастанием сложности.

4.4. Модель генерации дистракторов

Алгоритм
JavaScript 
// Модель создания неправильных вариантов ответа
function generateOptions(correctVerb) {
    const wrongOptions = allVerbs
        .filter(v => v.verb !== correctVerb.verb)  // Исключаем правильный
        .map(v => v.verb);
    
    const selectedWrong = shuffleArray(wrongOptions).slice(0, 3);
    return shuffleArray([...selectedWrong, correctVerb.verb]);
}

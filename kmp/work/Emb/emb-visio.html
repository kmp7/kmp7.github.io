Назначение

Интерактивное одностраничное приложение (SPA) предназначено для студентов-лингвистов, изучающих дистрибутивную семантику и векторные представления слов (word embeddings). Приложение наглядно сопоставляет два подхода к описанию значения слова:

  • Традиционный лексикографический — дискретный перечень значений с дефинициями, примерами употребления и тематическими пометами.
  • Дистрибутивно-семантический — непрерывное многомерное пространство, где каждое употребление слова представлено числовым вектором (эмбеддингом), а совокупность употреблений образует «семантическое облако».

Функциональность

Компонент Описание
Селектор слов Два предзаданных многозначных слова (bank, star). При выборе обновляются лексикографическая карточка и визуализация.
Лексикографическая карточка Пронумерованные словарные значения, контекстные примеры, тематические теги. Имитирует статью толкового словаря.
2D-визуализация Проекция идеализированного семантического пространства на плоскость. Каждый кластер значений представлен гауссовым эллипсоидом (1σ и 2σ), а отдельные употребления — точками-сэмплами.
Переключатель режимов «Идеальное (Gaussian)» — аналитические эллипсоиды нормального распределения; «Эмпирическое (корпус)» — облако точек с добавленным шумом, имитирующим разброс реальных корпусных эмбеддингов.
Режим сравнения Наложение семантических областей двух слов на одну плоскость. Визуализируется зона перекрытия, пунктирная линия между центроидами, подпись косинусного расстояния.
Метрики Косинусное расстояние между центрами облаков, коэффициент перекрытия (Bhattacharyya), число словарных значений, дисперсия семантического облака.
Интерактивные подсказки Наведение курсора на точку-сэмпл на канвасе показывает контекст (словосочетание), которому соответствует данный вектор.
Тёмная тема Переключение светлой / тёмной цветовой схемы кнопкой в заголовке. Все элементы, включая канвас, адаптируются к выбранной теме.
Адаптивность Интерфейс корректно отображается на настольных мониторах, планшетах и мобильных устройствах благодаря CSS Grid и медиазапросам.

Данные и модель

Координаты кластеров, сэмплов и метрики в этом демонстрационном приложении являются идеализированными и дидактическими. Они подобраны вручную так, чтобы наглядно иллюстрировать ключевые свойства эмбеддингов:

  • Полисемия отражается в нескольких разнесённых кластерах одного слова.
  • Гауссово распределение моделирует типичное допущение о форме семантического облака.
  • Эмпирический шум имитирует дисперсию, наблюдаемую в реальных контекстуализированных эмбеддингах (например, BERT, ELMo).
  • Косинусное расстояние и коэффициент Бхаттачарья предварительно заданы для пары слов и служат иллюстрацией, а не результатом вычислений над реальными векторами.

Для интеграции с реальными данными потребуется бэкенд-сервис, возвращающий координаты проекции (t-SNE / UMAP / PCA) корпусных эмбеддингов и метрики.

Лингвистический контекст

Приложение иллюстрирует переход от дискретной модели лексического значения (один лексема = конечный набор отдельных значений-лексико-семантических вариантов) к континуальной модели, принятой в дистрибутивной семантике, где:

  • Значение слова определяется его дистрибуцией — совокупностью контекстов употребления (гипотеза Дж. Р. Фёрса: «You shall know a word by the company it keeps»).
  • Каждое употребление представляется вектором в многомерном пространстве; семантически близкие употребления оказываются геометрически близкими.
  • Границы между значениями размыты: вместо чётких категорий — градиентные переходы и зоны перекрытия.
  • Гауссово приближение — одна из стандартных моделей для анализа формы семантического облака; на практике распределения могут быть мультимодальными и негауссовыми.

Что не сделано:

  • Подключение реального бэкенда с моделью эмбеддингов (BERT, GPT, fastText) и проекциями t-SNE / UMAP.
  • Ввод произвольного слова пользователем.
  • Выбор языка и корпуса (RNC, BNC, Wikipedia и др.).
  • Визуализация диахронного сдвига значений (сравнение эпох).
  • Добавление третьего слова для многостороннего сравнения.
  • Интерактивное исследование: перетаскивание кластеров, изменение дисперсии.
  • Экспорт визуализации в SVG / PNG для включения в учебные материалы.

Использованные технологии

HTML5
Семантическая разметка, элемент <canvas> для 2D-графики
CSS3
CSS Custom Properties (переменные) для тем, CSS Grid и Flexbox для раскладки, медиазапросы для адаптивности, анимации (keyframes, transitions)
JavaScript (ES6+)
Вся логика приложения — без фреймворков и внешних зависимостей. Canvas 2D API для отрисовки эллипсоидов, точек, сетки и оверлеев. requestAnimationFrame для плавных анимаций появления
SPA-архитектура
Единственный HTML-файл, без перезагрузки страницы. Данные хранятся в объекте JavaScript; переключение слов и режимов обновляет DOM и канвас динамически

Подсказка. В традиционной лингвистике слово описывается дискретным набором значений (лексем). В пространстве эмбеддингов каждое употребление слова — точка в непрерывном многомерном пространстве. Гауссов эллипсоид показывает идеализированное распределение; эмпирическое облако — реальные контексты из корпуса. Косинусное расстояние между центрами и степень перекрытия позволяют количественно сравнить семантическую близость двух слов.

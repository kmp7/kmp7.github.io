Редактор текста

Инструкция по работе

  • Введите свои ФИО и номер группы в соответствующие поля.
  • Нажмите "Продолжить" для перехода к редактору текста.
  • Используйте панель инструментов для форматирования текста.
  • Вставка текста из буфера обмена запрещена, ввод только с клавиатуры.
  • Нажмите кнопку "Сохранить", чтобы сохранить документ в формате PDF.
